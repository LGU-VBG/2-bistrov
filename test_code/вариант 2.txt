1. Инициализация массивов:

   • notes: массив значений банкнот.

   • count: массив, определяющий количество каждой банкноты.

2. Ввод данных:

   • Пользователь вводит сумму покупки и сумму наличных.

3. Расчет сдачи:

   • Если сумма сдачи отрицательная (т.е. покупатель дал меньше, чем стоит товар), выводится сообщение о недостатке средств.

   • Если сдача меньше 10 рублей, выводится сообщение без указания банкнот.

4. Подсчет банкнот:

   • Цикл проходит по массиву notes, вычисляя максимальное количество банкнот, которые можно использовать для выдачи сдачи.

   • Если остаётся сумма, которая не может быть выдана с помощью имеющихся банкнот, выводится сообщение об этом.
