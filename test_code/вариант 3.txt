1. Матрица: В примере задана матрица matrix с некоторыми положительными и отрицательными элементами.

2. Списки: Используются два списка: negativeElements для хранения всех отрицательных элементов и negativeRows для хранения индексов строк, состоящих только из отрицательных элементов.

3. Перебор: Код перебирает все элементы матрицы, добавляя отрицательные элементы в первый список и проверяя каждую строку на наличие хотя бы одного неотрицательного элемента.

4. Вывод: В конце выводятся найденные отрицательные элементы и номера строк с полностью отрицательными элементами. Если таких строк нет, выводится соответствующее сообщение.
